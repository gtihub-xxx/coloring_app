# Changelog

このプロジェクトへのすべての重要な変更はこのファイルに記録されます。

## [1.0.4] - 2026-01-02

### 追加
- 塗りつぶし範囲制限機能を追加
- クリックした場所から指定した範囲（0〜500px）のみを塗りつぶし可能に
- 0 = 無制限（従来通り）、1〜500 = 指定範囲のみ
- スライダーで簡単に範囲を調整
- 空間的につながっている場所でも、指定範囲のみを塗ることが可能

## [1.0.3] - 2026-01-02

### 修正
- アンドゥ機能のバグを修正
- 消しゴム使用後にアンドゥすると全て消えてしまう問題を解決
- 最初の描画状態に正しく戻れるように改善

## [1.0.2] - 2026-01-02

### 追加
- カスタムカラーピッカー機能を追加
- 16色のプリセットに加えて、任意の中間色を選択可能に
- 選択中の色コード（HEX）を表示

## [1.0.1] - 2026-01-02

### 修正
- 塗りつぶし機能の色許容範囲を10から50に拡大
- 塗りつぶしの最大反復回数を50,000から200,000に増加
- JPEG圧縮などによる色の微妙な変化に対応し、領域全体を正しく塗りつぶせるように改善
- 大きな領域でも塗りつぶしが途中で止まらないように改善

### 追加
- ヘッダーにバージョンバッジ (v1.0.1) を表示

## [1.0.0] - 2026-01-02

### 追加
- Webブラウザで直接動作する完全なぬりえアプリケーション
- 画像アップロード機能
- ブラシツール（可変サイズ: 2-50px）
- 塗りつぶしツール（フラッドフィルアルゴリズム）
- 消しゴムツール
- 16色のカラーパレット
- アンドゥ/リドゥ機能（最大20履歴）
- キーボードショートカット（Ctrl+Z/Cmd+Z でアンドゥ、Ctrl+Y/Cmd+Shift+Z でリドゥ）
- 描画した画像のダウンロード機能
- タッチデバイス対応
- 線画保護機能（明度閾値: 180）

### 技術仕様
- React 18 (UMD via CDN)
- ReactDOM 18 (UMD via CDN)
- Babel Standalone 7 (JSXトランスパイル)
- Lucide Icons (UMD via CDN)
- HTML5 Canvas API
- ビルドツール不要で動作

### 修正
- 色選択時にツールが保持されるように修正
- 塗りつぶし機能の変数名衝突を解消
- 塗りつぶしデータの更新ロジックを改善し、効率的なバッチ描画を実装
