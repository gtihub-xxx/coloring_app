# 🎨 ぬりえスタジオ (Coloring App)

**バージョン:** 1.0.1

Webブラウザで直接動作するぬりえアプリケーションです。ビルドツール不要で、HTMLファイルを開くだけで使用できます。

## ✨ 主な機能

- 🖼️ **画像アップロード**: 線画やぬりえ用の画像を読み込み
- 🖌️ **ブラシツール**: サイズ調整可能（2-50px）
- 🪣 **塗りつぶしツール**: フラッドフィルアルゴリズムで領域を一括塗りつぶし
- 🧹 **消しゴムツール**: 描画した部分を削除
- 🎨 **16色カラーパレット**: 豊富な色の選択肢
- ⏪ **アンドゥ/リドゥ**: 最大20回の操作を取り消し・やり直し
- ⌨️ **キーボードショートカット**:
  - `Ctrl+Z` / `Cmd+Z`: アンドゥ
  - `Ctrl+Y` / `Cmd+Shift+Z`: リドゥ
- 💾 **画像保存**: 描画した作品をPNG形式でダウンロード
- 📱 **タッチ対応**: タブレットやスマートフォンでも使用可能
- 🛡️ **線画保護**: 線画部分は塗りつぶされないように自動保護

## 🚀 使用方法

1. `index.html` をWebブラウザで開く
2. 「画像を選択」ボタンから線画をアップロード
3. カラーパレットから色を選択
4. ツールを選んで描画開始！
5. 完成したら「保存」ボタンで作品をダウンロード

## 🛠️ 技術スタック

- **React 18** - UIフレームワーク（CDN経由）
- **ReactDOM 18** - DOMレンダリング（CDN経由）
- **Babel Standalone** - JSXトランスパイル（CDN経由）
- **Lucide Icons** - アイコンライブラリ（CDN経由）
- **HTML5 Canvas API** - 描画機能

## 📋 必要な環境

- モダンなWebブラウザ（Chrome, Firefox, Safari, Edgeなど）
- インターネット接続（CDNからライブラリを読み込むため）
